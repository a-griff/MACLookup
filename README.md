# MACLookup

Version: 1.0
-------------------------------------------------------
Description:
Lookup manufacturer names based on MAC address prefixes.
For *NIXs, not for Windows.
-------------------------------------------------------
MACLookup.sh - Shell script for *NIX machines.

MACLookup.bas - FreeBASIC version.
- Requires curl,grep,head so this will not work on Windows.
- Run "fbc MACLookup.bas" to build the executable.
- Creates ./MACLookup.conf which contains user settings

MACLookup.pl - Perl version.
